# Auto Marketplace - Demo Spring Boot project

Java 17, Spring Boot 2.7.x, Thymeleaf, Bootstrap (webjars).

Run locally:

1. Install Java 17 and Maven.
2. Unzip project (if zipped).
3. From project root run:
   mvn spring-boot:run

Open: http://localhost:8080

This demo uses an in-memory list as a data store (no DB). Images referenced under /images are just example paths; you can place images into src/main/resources/static/images/.
